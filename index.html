☁ Base de datos en la nube

Conecta Firebase para que todos los empleados compartan los mismos datos en tiempo real desde cualquier dispositivo.

📌 Pasos en Firebase (gratis, sin tarjeta): 1. Entra en console.firebase.google.com y abre tu proyecto
2. Menú izquierdo: Compilación → Realtime Database → Crear base de datos
3. Ubicación Europa → Iniciar en modo de prueba → Habilitar
4. Icono ⚙ → Configuración del proyecto
5. Baja a "Tus apps" → pulsa </> (Web) → registra la app
6. Copia el bloque firebaseConfig entero y pégalo arriba
⚠ La app debe estar abierta desde una URL (https://…), no desde el archivo, para que funcione la conexión.
